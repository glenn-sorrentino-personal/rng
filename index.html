RNG source

Hardware TRNG mode reads raw bytes from a connected serial entropy device. Browser mode uses crypto.getRandomValues(), a cryptographically secure PRNG seeded by system entropy.

Run protocol

Select the target and run length before starting. These settings are locked during each run to reduce optional stopping and mid-run protocol changes.

How to read it

In a fair binary RNG, expected ones are n / 2. The z-score is (ones - n/2) / sqrt(n/4). Final p-values are most meaningful when the run length was chosen before the run.
